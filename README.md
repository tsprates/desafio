# Desafio

Segue as instruções abaixo para configurar e executar cada parte do projeto:

## API

```bash
cd api/
npm install
npm run serve
```

> Por padrão, a __API__ será executada na porta __*8000*__.

## UI

```bash
cd ui/
npm install
npm run dev
```

> Por padrão, a __UI__ será executada na porta __*5173*__.
